Typography

  • Base size: 1rem
  • Heading (h1): 2.1rem, 800 weight
  • Heading (h2): 1.3rem, 700 weight
  • Navigation: 1.1rem, 700 weight

Grid Item 2

  <div>Grid Item 1</div>
  <div>Grid Item 2</div>
</div>
  • .grid creates a CSS grid with gap
  • .grid-2, .grid-3 for columns
  • Margin/padding utilities: .mt-lg, .mb-lg, .mt-md, .mb-md

Images

        
<img src="..." alt="...">
  • Images have border-radius and shadow by default
  • Always use descriptive alt attributes

Accessibility

  • Focused elements have a clear outline using :focus
  • Buttons and links have visible hover/focus states
  • Use semantic HTML for headings, nav, lists, etc.
        
:focus {
  outline: 2px solid var(--color-link);
  outline-offset: 2px;
}

How to Use

  • Reference assets/css/design-system.css in your HTML files:
  • <link rel="stylesheet" href="assets/css/design-system.css">
  • Apply classes as documented above for components and utilities
  • Extend with new variables or components as your site grows!
